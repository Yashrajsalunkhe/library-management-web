# Library Management System

A comprehensive library management application built with Electron, React, and SQLite with biometric integration support.

## Features

- **Member Management**: Register and manage library members
- **Book Lending**: Track book loans and returns  
- **Attendance Tracking**: Biometric device integration for attendance
- **Payment Tracking**: Monitor membership fees and payments
- **Expenditure Management**: Track library expenses
- **Reporting**: Generate comprehensive reports
- **Email Notifications**: Automated email alerts
- **Data Backup**: Backup and restore functionality

## System Requirements

### Minimum Requirements
- **Operating System**: Windows 10 (64-bit) or later
- **RAM**: 4 GB
- **Storage**: 1 GB free space
- **Network**: Internet connection for email features

### Recommended Requirements
- **Operating System**: Windows 11 (64-bit)
- **RAM**: 8 GB or more
- **Storage**: 2 GB free space
- **Network**: Broadband internet connection

### Optional Hardware
- **Biometric Device**: ZKTeco or compatible biometric attendance device
- **Printer**: For generating reports and receipts

## Installation

1. Download the installer from the releases page
2. Run `Library-Management-System-Setup.exe`
3. Follow the installation wizard
4. Launch the application from the desktop shortcut

## First Run Setup

1. **Admin Account**: Create an admin account on first launch
2. **Email Configuration**: Configure SMTP settings for notifications
3. **Biometric Setup**: Connect and configure biometric device (optional)
4. **Library Settings**: Set up library information and operating hours

## Troubleshooting

### Common Issues

**Application won't start**
- Ensure Windows 10 or later
- Check if antivirus is blocking the application
- Run as administrator if needed

**Biometric device not detected**
- Check USB connection
- Install device drivers
- Verify device compatibility

**Email notifications not working**
- Check SMTP settings
- Verify network connection
- Check email provider settings

### Getting Help

1. Check the built-in help documentation
2. Review error logs in the application data folder
3. Contact support for technical assistance

## Uninstallation

1. Use Windows Add/Remove Programs
2. Or run the uninstaller from Start Menu > Library Management System
3. Choose whether to keep or remove application data
